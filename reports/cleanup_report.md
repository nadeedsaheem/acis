# Repository Cleanup Report

## Essential Files (KEEP)

These files are strictly required to reproduce the dataset, validate the output, and understand the extracted ACIS include structure.

| File Name | Purpose | Action | Reason |
| :--- | :--- | :--- | :--- |
| `multi_repo.py` | Main production C++ parser. | **KEEP** | Core script to generate the dataset. |
| `code_base.json` | Final parsed dataset. | **KEEP** | The primary deliverable containing all extracted functions, methods, and classes. |
| `verify_dataset.py` | Standalone verification script. | **KEEP** | Essential for automated precision and recall checks on the dataset. |
| `parse_summary.json` | Parser execution metrics. | **KEEP** (Move to `reports/`) | Necessary metric report verifying 100% file coverage. |
| `verification_report.json` | Verification output metrics. | **KEEP** (Move to `reports/`) | Essential record documenting class/inheritance validation runs. |
| `root_cause_analysis.md` | Investigation documentation. | **KEEP** (Move to `reports/`) | Explains why macro-based false positives appear in naive verifications. |
| `ACIS/` | Source code directory. | **KEEP** | The raw data source representing the ACIS include files. |
